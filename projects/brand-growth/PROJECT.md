# Brand Growth

This project covers brand, marketing, founder presence, and audience growth for Community Wolf.

Use this project for:

- brand system and messaging work
- content calendars and campaigns
- founder social planning and approvals
- launch announcements and cross-functional marketing tasks

Typical issue shape:

- one issue per campaign, content package, or measurable initiative
- subtasks for channel-specific execution when needed

Workspace model:

- keep source materials, content drafts, and reference docs inside each agent's own `workspace/` folder
- share approved outputs through Paperclip tasks, comments, and linked documents

Project structure:

- `briefs/` - active launch and campaign briefs
- `campaigns/` - initiative-specific working docs
- `calendars/` - current content and publishing calendars

Start here:

- `campaigns/community-wolf-social-content-strategy.md` - current company-level social strategy

Relationship to shared context:

- long-lived company and brand context belongs in `company/`
- active brand-growth work belongs in this project folder
