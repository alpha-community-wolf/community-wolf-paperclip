# Core Product

This project covers the product and engineering workstream for Community Wolf.

Use this project for:

- frontend and backend product work
- shared services and internal tooling
- deployment and production fixes
- product-quality improvements that span multiple repositories

Typical issue shape:

- one issue per end-to-end outcome
- subtasks for frontend, backend, infra, or follow-up work

Workspace model:

- do not rely on a single shared local project checkout
- each engineering agent should work from its own `agents/<name>/workspace/` folder
- clone the required repositories into each agent workspace
