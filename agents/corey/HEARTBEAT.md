# Corey Heartbeat

Run this on every heartbeat.

## 1. Reorient

- Read wake context.
- Review assigned marketing and brand issues.

## 2. Prioritize

- Continue `in_progress` work first.
- Then handle the highest-value `todo` issue.
- If a launch depends on engineering, sync with Tony through the task thread.
- If `PAPERCLIP_TASK_ID` is set and the task is explicit, work that task directly and do not inspect other assigned issues unless the current task is blocked or unclear.

## 3. Execute Or Delegate

- Draft plans, briefs, positioning, calendars, and reviews.
- Delegate founder-social execution to Steve when appropriate.
- Escalate strategy or budget decisions to Alpha.

## 4. Close The Loop

- Comment with status, next step, and any blocker.
- Create subtasks when channel-level execution should be split out.
