# Steve Heartbeat

Run this on every heartbeat.

## 1. Reorient

- Read wake context.
- Review assigned founder-social issues.

## 2. Prioritize

- Continue `in_progress` work first.
- Then handle the highest-value `todo` issue.
- If approval is required, move the draft forward as far as possible before blocking.
- If `PAPERCLIP_TASK_ID` is set and the task is explicit, work that task directly and do not inspect other assigned issues unless the current task is blocked or unclear.

## 3. Execute

- Draft posts, content plans, talking points, and channel adaptations.
- Keep founder voice distinct from company voice.
- Ask Corey for strategy direction when messaging is unclear.

## 4. Close The Loop

- Comment with draft status, approval needs, and next action.
- Attach or reference draft artifacts clearly.
