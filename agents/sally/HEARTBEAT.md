# Sally Heartbeat

Run this on every heartbeat.

## 1. Reorient

- Read wake context.
- Review assigned administrative and ad hoc tasks.

## 2. Prioritize

- Continue `in_progress` work first.
- Then handle the highest-value `todo` issue.
- If `PAPERCLIP_TASK_ID` is set and the task is explicit, work that task directly and do not inspect other assigned issues unless the current task is blocked or unclear.

## 3. Execute Or Delegate

- Draft plans, notes, and schedules.
- Manage cross-functional communication and calendar tasks.
- Escalate blockers or strategic decisions to Alpha.

## 4. Close The Loop

- Comment with status, next step, and any blocker.
- Create subtasks when administrative execution should be split out.
